# 📱 Инструкция по настройке Web App для MacTabak Bot

## ✅ Что готово:

1. **webapp.html** - полноценный веб-интерфейс каталога с:
   - ✅ 12 категорий с горизонтальной прокруткой
   - ✅ Фильтры (по цене ↑↓, новинки)
   - ✅ Поиск товаров с лупой
   - ✅ Визуальная корзина с счетчиком в правом нижнем углу
   - ✅ Управление количеством товаров (+/-)
   - ✅ Поле для промокода
   - ✅ Кнопка "Очистить корзину"
   - ✅ Закругленные края у карточек товаров

2. **bot.py** - обновлен для работы с Web App:
   - Кнопка "🛍 Открыть каталог" теперь открывает Web App
   - Обработка данных из Web App
   - Автоматический переход к оформлению заказа

## 🚀 Варианты размещения Web App:

### Вариант 1: GitHub Pages (Бесплатно)

1. Создайте репозиторий на GitHub
2. Загрузите файл `webapp.html`
3. Включите GitHub Pages в настройках репозитория
4. Получите URL вида: `https://yourusername.github.io/mactabak-webapp/webapp.html`
5. Замените `WEBAPP_URL` в bot.py на ваш URL

### Вариант 2: Netlify (Бесплатно)

1. Зайдите на https://app.netlify.com
2. Перетащите папку с `webapp.html`
3. Получите URL вида: `https://amazing-site-123.netlify.app`
4. Замените `WEBAPP_URL` в bot.py

### Вариант 3: Vercel (Бесплатно)

1. Установите Vercel CLI: `npm i -g vercel`
2. В папке с webapp.html выполните: `vercel`
3. Следуйте инструкциям
4. Получите URL и обновите bot.py

### Вариант 4: Cloudflare Pages (Бесплатно)

1. Зайдите на https://pages.cloudflare.com
2. Создайте проект, загрузите webapp.html
3. Получите URL вида: `https://mactabak.pages.dev`
4. Обновите WEBAPP_URL в bot.py

### Вариант 5: Локальный HTTPS (для тестирования)

1. Установите ngrok: `brew install ngrok` (Mac) или скачайте с https://ngrok.com
2. Запустите локальный сервер: `python3 -m http.server 8000`
3. В другом терминале: `ngrok http 8000`
4. Используйте временный HTTPS URL от ngrok

## 📝 Пошаговая инструкция (самый простой способ - Netlify):

### Шаг 1: Подготовка файла
```bash
# Создайте папку для веб-приложения
mkdir webapp
cp webapp.html webapp/index.html
```

### Шаг 2: Размещение на Netlify
1. Откройте https://app.netlify.com
2. Нажмите "Add new site" → "Deploy manually"
3. Перетащите папку `webapp` в браузер
4. Дождитесь загрузки (5-10 секунд)
5. Скопируйте полученный URL

### Шаг 3: Обновление бота
```python
# В файле bot.py замените:
WEBAPP_URL = "https://ваш-сайт.netlify.app"
```

### Шаг 4: Перезапуск бота
```bash
# Остановите бота (Ctrl+C)
# Запустите снова
python3 bot.py
```

## ⚙️ Настройка в @BotFather:

1. Откройте @BotFather в Telegram
2. Отправьте `/mybots`
3. Выберите вашего бота
4. Нажмите "Bot Settings"
5. Нажмите "Menu Button"
6. Отправьте URL вашего Web App

## 🎨 Кастомизация внешнего вида:

В файле `webapp.html` вы можете изменить:

- **Цвета**: Найдите `background: #333` и замените на ваш цвет
- **Закругления**: Найдите `border-radius: 12px` для изменения радиуса
- **Размер карточек**: Измените `grid-template-columns: repeat(2, 1fr)` на 3 для 3 колонок
- **Иконки товаров**: В массиве `products` замените эмодзи на свои

## ✅ Проверка работы:

1. Откройте бота в Telegram
2. Нажмите кнопку "🛍 Открыть каталог"
3. Должен открыться веб-интерфейс с товарами
4. Добавьте товары в корзину
5. Нажмите "Перейти к оформлению"
6. Данные автоматически передадутся в бот

## 🔧 Возможные проблемы:

**Проблема**: Web App не открывается
- Проверьте, что URL начинается с `https://`
- Убедитесь, что файл доступен по указанному URL

**Проблема**: Данные не передаются в бот
- Проверьте консоль браузера (F12)
- Убедитесь, что используете последнюю версию Telegram

**Проблема**: Стили не применяются
- Очистите кэш браузера
- Проверьте, что CSS правильно подключен

## 📞 Поддержка:

1. Правильность URL в bot.py
2. Доступность webapp.html по HTTPS
3. Актуальность версии Telegram
