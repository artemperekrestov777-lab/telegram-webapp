# MacTabak Shop Bot

## 🚀 Быстрый старт

### Установка зависимостей:
```bash
pip3 install -r requirements.txt
```

### Добавьте QR-код для оплаты:
Сохраните файл с QR-кодом как `payment_qr.jpg` в этой папке

### Запуск бота:
```bash
python3 bot.py
```

## ✅ Что уже создано:

### Структура проекта:
- `bot.py` - основной файл бота с полным функционалом
- `products_db.json` - база данных товаров
- `requirements.txt` - зависимости Python
- `payment_qr.jpg` - **НУЖНО ДОБАВИТЬ ВАШ QR-КОД!**

### Функционал бота:

#### Для клиентов:
- 🛍 **12 категорий товаров** (42 товара)
- 🛒 **Корзина** с автоматическим подсчётом суммы
- 📦 **Оформление заказа** с выбором доставки
- 💳 **Два способа оплаты:**
  - Для Москвы - через менеджера
  - Для других городов - через QR-код
- 📋 **История заказов**

#### Для администратора:
- ➕ Добавление товаров
- ✏️ Редактирование цен и остатков
- ❌ Удаление товаров
- 📊 Статистика продаж
- 📨 Массовая рассылка
