# MacTabak Bot - Telegram магазин табачной продукции

## Описание
Telegram бот для продажи табачной продукции с системой заказов, корзиной и разными способами доставки.

## Функционал
- 📱 Полноценный каталог с 12 категориями и 42 товарами
- 🛒 Корзина с управлением количеством товаров
- 📦 Оформление заказов с валидацией данных
- 🚚 Два способа доставки (СДЭК и Почта России)
- 💳 Разная логика для Москвы и регионов
- 👨‍💼 Админ-панель для просмотра заказов
- 📊 Статистика продаж

## Процесс заказа

### Для Москвы:
1. Выбор товаров → Корзина
2. Оформление → Ввод данных
3. Заказ отправляется менеджеру (ID: 706541536)
4. Менеджер связывается с клиентом

### Для регионов:
1. Выбор товаров → Корзина
2. Оформление → Ввод данных
3. Выбор доставки (СДЭК 500₽ / Почта 600₽)
4. Ввод адреса
5. Получение QR-кода для оплаты
6. Отправка чека на chek_qr@bk.ru
